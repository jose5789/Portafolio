Hola, soy Jose, soy un Desarrollador Junior lleno de pasión

Soy una persona enfocada en el desarrollo y mantenimiento de aplicaciones web creadas con PHP, tengo conocimientos avanzados en manejo de MYSQL, HTML5, CSS, Laravel, Wordpress, Cake php, Bootstrap y Git.

Se operar con estructuras de control condicionales y bucles; programando mediante el uso de sesiones, arreglos e includes; manejo de sistemas de archivos, subidas de archivos al servidor, ficheros y directorios.

Cuento con conocimiento de diseño de las bases de datos relacionales, tengo habilidades en el trabajo con tablas en sus diferentes tipos y restricciones; poseo habilidades en DML insert, update, select y delete; se operar insert avanzados, select avanzado, funciones generales, matemáticas, fechas, entre otras. Sumado al manejo de subconsultas multitabas y joins.

He programado diferentes aplicaciones usando el patrón de diseño MVC a través de la programación orientada a objetos a través del uso de PHP y sus diferentes complementos entre ellos un Administrador de pequeños negocios, Tienda virtual y blogs especializados.

A nivel personal soy una persona autodidacta apasionado por adquirir conocimientos con capacidades para adaptarme y aprender fácilmente, Poseo capacidad para trabajar en equipo y bajo presión. Soy una persona innovadora, proactiva comprometida con el trabajo, contribuyendo al bienestar general de la organización; soy una persona integral tanto en lo personal como en lo laboral.

Mis Habilidades

A continuación hare un breve recuento de las habilidades que poseo en el desarrollo de aplicaciones web.

HTML5

A través de este lenguaje se realizar la definición de una estructura básica para una aplicación web, además del manejo tablas, botones, formularios, listas, imágenes, diferentes etiquetas de tipo texto, videos entre otras etiquetas de este lenguaje de hipertexto.

CSS

Con este lenguaje de diseño grafico sé asignar deferentes estilos a los elementos con el fin de hacer paginas con un acabado más exacto a los diferentes requerimientos como ajustes de colores, formas, márgenes, tipos de letras y la posibilidad de crear diferentes versiones dependiendo el tamaño de los dispositivos.

PHP

Usando este lenguaje de programación tengo conocimientos en manejo de constantes, variables, operadores aritméticos, deferentes métodos de recepción de datos, implementación de condicionales, diferentes bucles (for, foreach, while, do while), uso de diferentes directorios, ficheros, subida de archivos y contenido audiovisual.

Git

Con este control de versiones podemos generar proyectos divergentes de un proyecto principal para diferentes propósitos para ello sé la implementación de diferentes comandos usados para generar comentarios en el código, descargar el mismo y también actualizarlo en la línea principal del proyecto o también en ramas diferentes del mismo a su vez.

Bootstrap

Al usar este framework css sé implementar diferentes estilos a la modificación de contenedores de información, tipografía, colores, tablas, imágenes, alertas, botones, grupos de botones, barras de navegación, formularios, carrusel de imágenes, tarjetas, barras de progreso, menús desplegables entre otros.

MYSQL

Con este sistema de gestión de bases de datos se realizar la creación de bases de datos, tablas, modificación de tablas y sus componentes manejo de inset, select, update, delete, where, consultas de agrupamiento, subconsultas, iner join, left join ,right join, generación de vistas entre otros.

Laravel

Instalación de proyecto de Laravel en distintas versiones, manejo de rutas, rutas con parámetros, parámetros opcionales, uso de la consola artisan y sus comandos if, bucles, includes manejo de una plantilla base o layout , manejo de controladores, modelos y vistas, enlaces, redireciones, formularios, tablas, listas migraciones, creación de crud entre otros.

Wordpress

Instalación de Wordpress manejo de la barra de herramientas, escritorio del panel, gestión de entradas y categorías, biblioteca de medios, ajustes de comentarios, medios, enlaces permanentes y herramientas; uso de diferentes plugins en función del requerimiento.

Cakephp

Instalación de proyecto de cake php en distintas versiones, uso de consola bake manejo de condicionales if, bucles, includes manejo de una plantilla base o layout , manejo de controladores, modelos y vistas, enlaces, redirecciones, formularios, tablas, listas, creación de crud entre otros entre otros.
